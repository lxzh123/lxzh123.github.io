      以下为输出结果：
      排序前:

1       7       7       0       5       2       1       3

      排序后:

1       7       0       5       2       1       3       7
1       0       5       2       1       3       7       7
0       1       2       1       3       5       7       7
0       1       1       2       3       5       7       7
0       1       1       2       3       5       7       7
0       1       1       2       3       5       7       7
0       1       1       2       3       5       7       7
0       1       1       2       3       5       7       7
0       1       1       2       3       5       7       7

      右上可以看到，每一轮排序，每行都会有个数字由前向后想冒泡一样移动，因此这种排序正是得名于像“气泡一样”从序列的一段浮动到另一端。

      以上为升序排序，若采用降序排序，则只需稍微改动一点点，即将BubbleSort方法中的“>”号改为“<”即可。
